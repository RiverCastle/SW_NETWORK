# 🎬 못 만드는 분과장의 생존기 (가제)

> **한 줄 컨셉:** 영상 편집 한 번 안 해본 사람이 어쩌다 미디어분과장이 되어, AI(Claude·ChatGPT·Gemini)와 분과원들의 도움으로 홍보영상을 만들며 겪는 좌충우돌 에피소드.

인스타 릴스 · 유튜브 숏폼 콘텐츠 기획서를 모아두는 저장소입니다.

## 📁 구조
화별로 디렉토리를 만들어 관리합니다.

```
SW_NETWORK/
├── README.md                ← 시리즈 개요 (이 파일)
└── 01화_도전의-시작/
    ├── 기획서.md            ← 기획 (훅·씬·사운드·CTA·촬영 체크리스트)
    ├── 콘티.md              ← 씬별 앵글·구도·소품 (촬영 현장용)
    ├── 대본.md              ← 촬영·녹음용 대사
    ├── 편집노트.md          ← 컷 순서·B롤·자막·BGM 구간 (편집용)
    ├── 회의록.md            ← 단계별 회의 아젠다·결정사항·액션아이템
    └── 업로드.md            ← 발행용 제목·캡션·해시태그·고정댓글
```

### 각 화 표준 구성 (6종)
| 파일 | 단계 | 용도 |
|---|---|---|
| `기획서.md` | 기획 | 전체 설계도 |
| `콘티.md` | 촬영 전 | 씬별 앵글·구도·소품, 현장 들고 다니는 용 |
| `대본.md` | 촬영·녹음 | 대사만 정리한 낭독용 (프롬프터·AI 더빙) |
| `편집노트.md` | 편집 | 컷 순서·B롤·자막 타이밍·BGM 구간 |
| `회의록.md` | 전 단계 | 기획→촬영→편집→발행 단계별 회의 기록 |
| `업로드.md` | 발행 | 인스타/유튜브 복붙용 묶음 |

2화부터도 같은 6종 구성을 따릅니다.
