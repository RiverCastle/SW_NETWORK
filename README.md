## 📁 구조
화별로 디렉토리를 만들어 관리합니다.

```
SW_NETWORK/
├── README.md                ← 시리즈 개요 (이 파일)
└── 01화_도전의-시작/
    └── 기획서.md
```

각 화 디렉토리에는 해당 화의 `기획서.md`가 들어 있고, 필요 시 대본·자막·레퍼런스 등을 같은 폴더에 추가합니다.
